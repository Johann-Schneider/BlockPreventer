## BlockPreventer

BlockPreventer is an advanced and open source plugin, that disables breaking blocks for all players. You can configure what gamemode you donâ€™t want players to break blocks.

Requirements:

- a Spigot/Paper Server
- Java 8+

Config:

```
# BlockPreventer Configuration (by Donut4GAMER)
# thanks for downloading this plugin

# set the gamemode, where the blockprventer works
# use: SURVIVAL, ADVENTURE, CREATIVE
Gamemode: SURVIVAL

# set the bypasspermission that allows users to break blocks even tho they are in the gamemode listed above
BypassPermission: "blockpreventer.bypass"
```

Setup:

1. Download the jar
2. Put it in your /plugins/ folder
3. Start/Restart your server
4. You are done
